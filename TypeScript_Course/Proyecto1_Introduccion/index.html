Proyecto 1 - Introduccion a TypeScript

Sección 1: Base introductoria

Se aprendió los conceptos básicos iniciales de TypeScript, como lo es la compilación de archivos, el uso de archivos JS en HTML y el uso de archivos TS. Se entendió que saber JavaScript es fundamental para aprender un 80% mas rápido TypeScript, dado que TypeScript es un superconjunto de JavaScript.

Sección 2: Modo Observador

Se aprendió a como inicilar un proyecto en TypeScript, para obtener muchas ventajas, como por ejemplo, el uso de la compilación de archivos TS a JS de manera automática. A esto se le conoce como el modo observador, que es un modo en el que el compilador de TypeScript se queda observando las modificaciones de los archivos TS en tiempo real y los compila automáticamente. Esto es muy útil para desarrollo de aplicaciones web.

Sección 3: Tipos de Datos

Se aprendió a como declarar variables en TypeScript. Se tienen varios tipos de datos, entre ellos:
* Primitivos: string, number, boolean, undefined, null
* Compuestos(Estructurado): object, array, enum, function
* Definidos por el usuario: class, interface, type
