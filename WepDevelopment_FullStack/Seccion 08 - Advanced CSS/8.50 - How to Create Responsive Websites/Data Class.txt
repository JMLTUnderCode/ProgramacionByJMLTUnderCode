Un termino importante en la programacion web es 
RESPONSIVE eso se define como aquella propiedad
que permite que tu pagina web sea vea bien y 
estructurada en todo dispositivo web, esto es
telefonos, tablet, laptops, computadora, etc.

Las formas para conseguir este display responsivo 
es por 

1. Media Query

@media (max-width: 600px){
    /* CSS code for screen below or equal to 600px wide*/
}

Dentro de ee bloque de codigo iran los comando asociados
al display de para una window menor o igual a 600px.

Este modelo permite modificar y agregar cuadriculas en
